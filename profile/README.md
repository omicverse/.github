## Welcome to [omicverse](https://github.com/omicverse)!

**omicverse** is a community-built open-source ecosystem for omics data analysis in Python — from foundational I/O and reimagined algorithms to end-user interfaces. Everything you see below is open to contribution, whether you bring a new algorithm, a tutorial, or a port of an R package.

Here's how to get started:

- 📖 Explore the [documentation](https://omicverse.readthedocs.io).
- 🎓 Walk through the [tutorials](https://omicverse.readthedocs.io/en/latest/Tutorials/).
- 📦 Install from [PyPI](https://pypi.org/project/omicverse) or [conda-forge](https://anaconda.org/conda-forge/omicverse).
- 📄 Read the [paper](https://doi.org/10.1038/s41467-024-50194-3) in *Nature Communications* (2024).
- 🤝 **Contribute** an algorithm, a tutorial, or an R-to-Python port — [open an issue](https://github.com/omicverse/omicverse/issues/new) or [start a discussion](https://github.com/omicverse/omicverse/discussions).
- 🌐 Visit our [website](https://omicverse.com).
